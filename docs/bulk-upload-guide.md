# 一括投稿 ファイル準備ガイド

## フォルダ投稿（推奨）

### 必要なもの

1. **目次ファイル**（`index.md`）― ファイル名とタイトルの対応表
2. **各話のファイル**（`.md`）― 1ファイル = 1話

### フォルダ構成

```
my-novel/
├── index.md         ← 目次（必須）
├── chapter01.md     ← 第1話の本文
├── chapter02.md     ← 第2話の本文
├── chapter03.md     ← 第3話の本文
└── ...
```

### 目次ファイルの書き方（index.md）

各行に **ファイル名**（スペース）**タイトル** を記述します。

```
chapter01.md 旅立ちの日
chapter02.md 出会い
chapter03.md 森の中で
chapter04.md 決意
chapter05.md 最終決戦
```

**ルール:**
- 上から順に第1話、第2話…と話数が割り振られます
- `#` で始まる行と空行は無視されます（コメントとして使えます）
- ファイル名にはスペースを含めないでください
- 目次ファイルの名前は `index.md` または `目次.md` にしてください

**コメント付きの例:**

```
# 第一部 冒険の始まり

chapter01.md 旅立ちの日
chapter02.md 出会い
chapter03.md 森の中で

# 第二部 対決

chapter04.md 決意
chapter05.md 最終決戦
```

### 各話ファイルの書き方

ファイルの中身は本文のみ。タイトルは目次ファイル側で指定するので不要です。
独自記法（ルビ・傍点・場面転換など）はそのまま使えます。

```
　朝靄の中、少年は村を出た。

「もう戻らない」

　{背嚢|はいのう}を背負い直し、..決意..を胸に一歩を踏み出した。

===

　街道を半日ほど歩くと、小さな町が見えてきた。
```

## 単一ファイル投稿

1つのファイルに全話をまとめる方式です。フォルダを用意する手間が不要です。

### 書き方

`# タイトル` （H1見出し）で各話を区切ります。

```markdown
# 第1話 旅立ちの日

　朝靄の中、少年は村を出た。
本文をここに書く...

# 第2話 出会い

本文をここに書く...

# 森の中で

「第○話」の番号部分は省略してもOK。
```

**ルール:**
- `# ` で始まる行が話の区切りになります
- `# 第N話 タイトル` と書いた場合、「第N話」部分は自動で除去されタイトルだけ使われます
- 最初の `# ` より前のテキストは無視されます

## 投稿手順

1. ダッシュボード → 対象の小説の「一括投稿」ボタン
2. 「フォルダで投稿」または「単一ファイルで投稿」タブを選択
3. ファイル/フォルダを選択
4. プレビュー画面でタイトル・話数を確認（編集・削除も可能）
5. 「○話を下書き保存」ボタンで投稿

投稿後はすべて**下書き状態**で保存されます。
章管理ページから個別または一括で公開できます。

---

## 制限事項

- 一度に投稿できるのは最大 **100話** まで
- ファイルは **UTF-8** エンコーディングのみ対応
- 対応拡張子: `.md` `.txt` `.markdown`
- フォルダ全体の合計サイズ上限: **50MB**
